Status

The Free BSD/ppc prot is still a Tier 2 platform. That means it is not being fully supported by our security officer, release engineers and toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC G5 has been commited to SVN.
  • 03 March, 2008: Support for Freescale® PowerQUICC III MPC85XX family system-on-chip development boards has been committed into CVS
  • 27 February, 2008: Free BSD 7.0 si the first release to offically support the FreeBSD/ppc port.
